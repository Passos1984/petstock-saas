# 🐾 PetStock - Gestão Inteligente para Agropets

O **PetStock** é um sistema SaaS (Software as a Service) focado na modernização de agropecuárias e casas de ração. Desenvolvido para oferecer agilidade no balcão e controle total de estoque, ele elimina a necessidade de equipamentos caros, transformando qualquer smartphone em uma ferramenta poderosa de venda e gestão.

## 🚀 Diferenciais do Sistema

* **PDV Mobile Integrado:** Realize vendas direto no corredor ou no balcão usando a câmera do celular como leitor de código de barras.
* **Gestão de Estoque em Tempo Real:** Cadastre produtos e dê entradas/baixas direto da prateleira, sem precisar ir até o computador.
* **Venda a Granel Simplificada:** Lógica inteligente para fracionamento de sacos de ração e venda por peso (KG) com cálculo automático.
* **Comprovante via WhatsApp:** Envie o recibo da venda instantaneamente para o cliente, economizando papel e modernizando o atendimento.
* **Sistema de Brindes:** Função rápida no PDV para bonificação de clientes com baixa automática no estoque.
* **Radar de Reposição:** Alertas inteligentes baseados no tempo de consumo médio para fidelização de clientes.

## 🛠️ Tecnologias Utilizadas

* **Backend:** Python com Flask
* **Frontend:** HTML5, CSS3 (Design Responsivo Premium) e JavaScript
* **Banco de Dados:** SQLite (Integrado via SQLAlchemy)
* **Integrações:** Biblioteca `html5-qrcode` para leitura de códigos EAN/SKU via câmera.

## 📸 Demonstração

O sistema foi desenhado com foco na experiência do usuário (UX), garantindo que mesmo operadores sem experiência técnica consigam dominar a ferramenta em poucos minutos.

> **Nota:** Os recursos de leitura de câmera e atalhos rápidos foram otimizados para navegadores modernos (Chrome/Safari) tanto em Android quanto iOS.

## 💻 Como Rodar o Projeto

1.  Clone o repositório:
    ```bash
    git clone [https://github.com/SEU_USUARIO/petstock.git](https://github.com/SEU_USUARIO/petstock.git)
    ```
2.  Instale as dependências:
    ```bash
    pip install -r requirements.txt
    ```
3.  Inicie o servidor:
    ```bash
    python app.py
    ```
